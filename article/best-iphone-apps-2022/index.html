移动端上一些可以折腾的APP
本文创作于2022年9月9日 11:04

可以折腾, 就意味值得用户花费大量时间, 所以它就需要具备一定的门槛且确切的解决用户痛点. 这就要求应用在自己限定的领域中具有全能力的可扩展性(自定义) , 下面非常主观的推荐一些非常具有可玩的应用.

★★★☆☆ / ¥98.00 / 欧陆词典 Pro

支持多种扩展词库的离线词典工具
平台: iPhone/iPad/Mac
免费版: YES

Pro版: 自定义语音库, AD

离线扩展词库下载

  • https://mdx.mdict.org
  • https://downloads.freemdict.com (推荐)

折腾技能树: html

★★★★☆ / ¥12.00 / Stay 2

Safari浏览器扩展, 用户脚本扩展管理
平台: iPhone/iPad/Mac
免费版: YES

Pro版: 网页暗黑模式/iCloud同步

脚本下载

  • https://openuserjs.org
  • https://greasyfork.org/zh-CN (推荐)

折腾技能树: javascript

★★★★☆ / ¥12.00 / AdGuard

Safari浏览器扩展, 摆脱恼人广告
平台: 全平台
免费版: YES

Pro版: 自定义过滤器

iOS端需要非国区app store帐号下载
推荐在淘宝购买 个人永久版 的授权码(3台设备), 绑定在官网注册的帐号上用于激活.

应用自带丰富的过滤规则, 下面是推荐的规则

  • https://anti-ad.net/adguard.txt
  • https://raw.githubusercontent.com/damengzhu/banad/main/jiekouAD.txt
AdGuard桌面端还附带用户脚本扩展管理的功能, 在相应浏览器上启用插件后可以共享脚本, 做到一次安装多处使用

折腾技能树: html/css/AdGuard规则

★★★★☆ / $4.99 / Loon

科学冲浪, 高性价比的网络调试工具
平台: iPhone/iPad/Mac
免费版: NO

telegram社群

  • https://t.me/LoonNews
  • https://t.me/LoonTutorial
  • https://t.me/Loon0x00

折腾技能树: html/javascript

★★★☆☆ / Free / Scriptable

支持运行JavaScript的widgets
平台: iPhone/iPad
免费版: YES

telegram社群

  • https://t.me/Scriptable

折腾技能树: javascript/Scriptable API文档

局域网多平台资源共享/剪贴板, 使用起来绝对没有毛病
平台: 全平台
免费版: YES

官网: kdeconnect.kde.org

折腾技能树: 无 (这款应用不折腾人, 但是可以更加轻松的在多平台上进行折腾)
