黑苹果搭建之旅 / HACKINTOSH:I7-9700K+RX580+Z390-ITX+NZXT-H200

本文创作于2019年7月10日 11:21

折腾黑苹果对于普通使用者是一件有啥意义的事情, 做这个选择的根本原因还是穷qióng呀!

笔者本身拥有一台2015款的13寸rMBP的定制版, 经过4年多的战斗后在处理代码编译或者视频渲染时已经捉襟见肘了. 所以根据自己的需求&荷包最终选择搭建一套黑苹果(实力不允许为所欲为呀).

硬件

需求

首先摆一下笔者的需求, 不然脱离需求谈装机都是耍流氓呀,

  • 预算1w以内(划重点)
  • 机箱不能太大, 放置和搬家都不方便
  • 代码编译速度足够快, 提升生产效率
  • 视频剪辑和渲染基本流畅

硬件选择和限制

组装黑苹果的原则: 硬件选对可以少走很多弯路

处理器/CPU

CPU优先或者必须是选牙膏厂的(intel), 关于不同型号的性能可以参考CPU天梯图. 笔者选择了i7-9700K其性能基本满足需求, 而且超频可以应付更多场景.

显卡/GPU

GPU优先选择A卡(amd),一般都是可以免驱的. 根据GPU天梯图和Mojave硬件支持列表备选显卡有:

由于机箱选择的是机箱选择了NZXT H200可以容纳的最大显卡高度需小于50mm, 就剩下Vega56, RX590, RX580, RX480, 在价格和性能间取了一个中间点, 选了RX580.

主板/motherboard

机身不能太大, 为了搭配i7-9700K使用也赶个时髦, 决定选择z390系列的ITX主板. 当下可以选择包括:

  • 技嘉(Gigabyte) Z390I AORUS PRO WIFI
  • 华擎(asrock) Z390 Phantom Gaming itx/ac
  • 微星(MSI) MPG Z390I GAMING EDGE AC

搜索网上黑苹果成功案例, 使用华擎和技嘉的主板比较多, 然后华擎便宜就选华擎了.

硬件搭建

这是我第一次搭建PC主机, 基本上就是一步步查教程搞定的.过程中我碰到一些需要注意的点记录如下:

  • 安装intel cpu
  • 全模组电源如何连线

  • BCM94360CS2无线模块(拆装视频)

软件搭建

系统安装和调教的大概流程如下:

  1. 制作引导U盘
  2. 通过U盘clover安装系统
  3. 在系统中安装clover configuration manager
  4. 生成serial code





Clover安装在哪个盘都没有关系，你的BIOS设置好就行了（安装在A硬盘就设置为A硬盘引导，安装在B就设置为B引导）。
我目前也是双硬盘，主硬盘安装的Clover+macOS，第二硬盘安装的Win10，Win10的引导自动安装在第二硬盘了，我BIOS设置为主硬盘引导，就能进入Clover了，Clover会扫描所有硬盘的UEFI引导，发现有Win10的也会列出来，启动的时候选择ESP_EFI这个区的就行了（Clover主题图标会显示为Windows的标志）
